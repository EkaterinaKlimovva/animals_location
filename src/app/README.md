# Animals Location App

Эта папка содержит основное приложение с настройкой подключения к базе данных и запуском сервера.

## Структура

- `app/index.ts` - Основной файл приложения с настройкой Express сервера
- `app/database.ts` - Настройка подключения к базе данных Prisma

## Запуск

### Для разработки
```bash
npm run dev:app
```

### Для продакшена
```bash
npm run start:app
```

## Особенности

- Автоматическое подключение к базе данных при запуске
- Graceful shutdown при SIGINT/SIGTERM
- Логирование статуса подключения и запуска сервера
